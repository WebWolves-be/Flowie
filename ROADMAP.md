# Flowie Roadmap

This document contains feature requests and improvements based on user feedback.

---

## Dashboard

| Feature | Description |
|---------|-------------|
| Private Tab | Ability to have a personal/private tab visible only to specific users (e.g., for hour tracking) |
| Color Customization | Option to change the primary blue color to other colors |
| Dashboard Content | Define what information/widgets should be displayed on the dashboard |

---

## Projects

| Feature | Description |
|---------|-------------|
| Combined View with Labels | In the "All" tab, show all projects together with a label indicating which entity they belong to (e.g., NOVARA or IMMOSEED) |
| General Tab | Add a new "General" tab for non-categorized items |
| Task Drag & Drop | Ability to drag and drop tasks to reorder them |
| Buyer Role (Inkoper) | Add a "Buyer" role for assigning coworkers (future consideration) |

---

## Tasks & Subtasks

| Feature | Description |
|---------|-------------|
| Reconsider Task/Subtask Division | Evaluate if the task/subtask hierarchy adds clarity or reduces overview |
| Optional Coworker Assignment | Make the coworker/assignee field optional instead of required |
| Optional Deadline | Keep deadline field visible but not required |
| Creation Date Display | Show when a task was created to track how long projects take from start to completion |
| List-Style Descriptions | Option to format task descriptions as bullet lists (vertical) instead of inline text |
| Task/Subtask Drag & Drop | Ability to drag and drop tasks and subtasks to reorder their position |

---

## Status & Workflow

| Feature | Description |
|---------|-------------|
| "Waiting On" Status | Add a new status for when an action is completed but waiting on a third party (e.g., waiting for certificate delivery) |
| Keep Completed Items Visible | Archive completed tasks/projects but keep them searchable and visible for reference and motivation |

---

## Attachments & Integrations

| Feature | Description |
|---------|-------------|
| File Attachments | Ability to attach files to tasks (via Magic Map integration) |
| Image Attachments | Ability to add images directly to tasks |
| Email Linking | Ability to link/attach emails to tasks |
| Whise Integration | Potential integration with Whise platform |
| Calendar Integration | Sync deadlines with external calendar applications |

---

## Notifications & Alerts

| Feature | Description |
|---------|-------------|
| Pop-up Notifications | Desktop/browser pop-up alerts for tasks and emails with progress updates |
| Priority Levels | Ability to set priority levels on tasks (e.g., low, medium, high, urgent) |

---

## User Experience

| Feature | Description |
|---------|-------------|
| Profile Photos on Assignments | Display profile photos on subtasks to easily identify assignees |
| Session Timeout | Extend or configure the inactivity timeout to prevent automatic logout during workday |

---

## Notes

- Priority levels will be determined in a future planning session
- Features are grouped by category for easier review
- "Medewerker" (coworker) functionality can be expanded later under the "Inkoper" (Buyer) role
